#

> Setup
```
bash .github/scripts/setup.sh
```

> Build & Execute
```
cd build
cmake ..
cmake --build .
cd ..

./build/bomberman
```

> Generate Doxygen
```
cmake -DDOXYGEN=ON ..
```

> Run Unit Tests
```
cmake -DUNIT_TEST=ON ..
...

./build/unit_tests

```
